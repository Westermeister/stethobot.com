Model
=====
- Type: Balanced Random Forest

Dataset Source
==============
- The model was fit on the following dataset:
  https://www.kaggle.com/uciml/pima-indians-diabetes-database

Dataset Predictors
==================
- The dataset contains the following 8 predictors:
    1. Number of times pregnant.
    2. Plasma glucose concentration, 2 hours in an oral glucose tolerance test.
    3. Diastolic blood pressure (mmHg).
    4. Skin thickness i.e. triceps skin fold thickness (mm).
    5. Insulin i.e. 2-hour serum insulin (muU/ml).
    6. BMI i.e. body mass index.
    7. Diabetes pedigree function.
    8. Age in years.
- The final column is the label.
    9. 1 for diabetes, 0 for no diabetes.
