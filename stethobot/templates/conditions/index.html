Breast Cancer

Statistics

  • Accuracy: 95%
  • Sensitivity: 94%
  • Specificity: 95%

Diabetes

Statistics

  • Accuracy: 86%
  • Sensitivity: 86%
  • Specificity: 86%

Sample Source

The diagnostic data used to fit the model originates from the National Institute of Diabetes and Digestive and Kidney Diseases.

The dataset was previously available at the UCI Machine Learning Repository, but has since been taken down for reasons unknown. However, a copy can still be accessed (last I checked: March 2021) via Kaggle.

NOTE: There are missing values (filled as zeros) starting from the Glucose column going through the Age column. Stethobot currently handles these missing values by imputing them via the label-wise median. An alternative - simply removing observations with missing values - was also attempted, but this led to a ~10% drop in accuracy compared to imputation.
